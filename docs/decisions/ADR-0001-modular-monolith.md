# ADR-0001

## Title

Use Modular Monolith

## Status

Accepted

## Context

BusinessOS will initially be developed by one engineer.

## Decision

Use a Modular Monolith architecture.

## Consequences

Positive

- Faster development
- Easier deployment
- Easier debugging

Negative

- Requires discipline between modules

Future

Can migrate modules into microservices when needed.
