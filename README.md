# BusinessOS

BusinessOS is the flagship enterprise SaaS platform developed by Dhanvix Technologies.

## Vision

Build modular business software that helps SMEs digitize their operations through secure, scalable and intelligent applications.

## Technology Stack

- Laravel
- Angular
- MySQL
- Redis
- Docker
- Swagger (OpenAPI)
- GitHub Actions
- AWS

## Current Status
